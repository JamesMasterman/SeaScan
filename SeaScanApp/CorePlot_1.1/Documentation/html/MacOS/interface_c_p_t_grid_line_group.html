Detailed Description

A group of grid line layers.

When using separate axis layers, this layer serves as the common superlayer for the grid line layers. Otherwise, this layer handles the drawing for the grid lines. It supports mixing the two modes; some axes can use separate grid line layers while others are handled by the grid line group.
