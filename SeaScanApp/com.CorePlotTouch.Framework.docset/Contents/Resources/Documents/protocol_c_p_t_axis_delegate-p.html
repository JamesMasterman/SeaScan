shouldUpdateAxisLabelsAtLocations: (NSSet *)  locations 

(Optional) This method gives the delegate a chance to create custom labels for each tick. It can be used with any labeling policy. Returning NO will cause the axis not to update the labels. It is then the delegate’s responsiblity to do this.
